# Draft Rubrik Penilaian Kinerja Guru

Rubrik supervisi/observasi kelas. **Skala skor tiap indikator: 1–4**
(1 = Kurang, 2 = Cukup, 3 = Baik, 4 = Sangat Baik)

Skor akhir dihitung berbobot:
`Skor akhir = Σ (skor indikator × bobot) / Σ bobot` → dinormalisasi ke skala 100 jika perlu.

> Catatan: ini draft awal. Sesuaikan kategori/indikator dengan standar sekolah/dinas (mis. mengacu pada instrumen PKG / supervisi akademik yang dipakai).

## Contoh perhitungan

Misal total bobot semua indikator = 100. Supervisor mengisi skor 1–4 tiap indikator.

- Skor maksimum = Σ(4 × bobot) = 4 × 100 = 400
- Skor diperoleh = Σ(skor × bobot)
- **Nilai akhir (0–100)** = (Skor diperoleh / 400) × 100

**Kategori hasil:**
- 86–100 → Sangat Baik
- 71–85 → Baik
- 56–70 → Cukup
- < 56 → Perlu Pembinaan
